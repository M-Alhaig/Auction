Setup Instructions

  2. Set the Authorized redirect URI to:

OAuth2 Flow

  1. Click "Login with Google/GitHub"
  2. Browser redirects to provider's authorization page
  3. You approve the app
  4. Provider redirects back here with authorization code
  5. This page sends code to backend: POST /api/auth/oauth2/google
  6. Backend exchanges code for tokens, returns JWT
