How to Test

  1. Make sure all services are running: User (8081), Item (8082), Bidding (8083), Notification (8084)
  2. Click Login to authenticate and get a JWT token
  3. Click Connect to establish WebSocket connection with JWT
  4. Enter an Item ID and click Subscribe to /topic/items/{id}
  5. Open a second browser tab and login with a different user
  6. Use Postman (or another tab) to place bids:
    • First bid: User 1 places $150 → Watch "Bid Updates" panel
    • Outbid: User 2 places $200 → Watch User 1's "Personal Alerts" panel
  7. Watch real-time notifications appear!
