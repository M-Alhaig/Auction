How to Test

  1. Make sure all services are running: Item (8082), Bidding (8083), Notification (8084)
  2. Click Connect to establish WebSocket connection
  3. Enter an Item ID and click Subscribe to /topic/items/{id}
  4. Open a second browser tab with a different User ID (e.g., change last digit)
  5. Use Postman to place bids:
    • First bid: Bidder 1 places $150 → Watch "Bid Updates" panel
    • Outbid: Bidder 2 places $200 → Watch Bidder 1's "Personal Alerts" panel
  6. Watch real-time notifications appear!
